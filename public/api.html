Bluzit API

Bluzit Web API

Our Web API lets application to fetch data from Bluzit music catalogue and manage user's playlist and saved music.

Based on simple REST principles, Our Web API endpoints return metadata in JSON format about artists, albums and songs directly from the Bluzit catalogue.

The API also provides access to user-related data such as genres saved in past visits in site, subject to user's authorization.

Instructions

Method: GET
Required Parameters: user_id <Number>
URL Example: https://ws-bluzit-servce.herokuapp.com/get_user_data/1
Response: get the user data from DB
