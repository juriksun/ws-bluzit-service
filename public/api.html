Bluzit API

Bluzit Web API

Our Web API lets application to fetch data from Bluzit music catalogue and manage user's playlist and saved music.

Based on simple REST principles, Our Web API endpoints return metadata in JSON format about artists, albums and songs directly from the Bluzit catalogue.

The API also provides access to user-related data such as genres saved in past visits in site, subject to user's authorization.

Instructions

Set New User

Method: POST
Required Parameters: username: <string>, password: <string>, first_name: <string> last_name: <string>, email: <string>, dob: <string>
URL Example: https://ws-node-bluzit-service.herokuapp.com/set_new_user/
Response: Set new user in DB.

Set User Genres

Method: POST
Required Parameters: username: <string>, password: <string>, genre_<Number>: <String>
URL Example: https://ws-node-bluzit-service.herokuapp.com/set_user_genres/
Response: Set User Genres in DB.

Reset User Genres

Method: POST
Required Parameters: username: <string>, password: <string>, genre_<Number>: <String>
URL Example: https://ws-node-bluzit-service.herokuapp.com/reset_user_genres/
Response: Delete all Exist Genres in User info and sets new picked Genres.

Set User Album

Method: POST
Required Parameters: username: <string>, password: <string>, album_id <Number>
URL Example: https://ws-node-bluzit-service.herokuapp.com/set_user_album/
Response: Set the user picked Albums to DB

Delete User Album

Method: POST
Required Parameters: username: <string>, password: <string>, album_id <Number>
URL Example: https://ws-node-bluzit-service.herokuapp.com/del_user_album/
Response: Delete the user picked album data from his profile in DB

Get User Verification

Method: POST
Required Parameters: username: <string>, password: <string>
URL Example: https://ws-node-bluzit-service.herokuapp.com/get_user_verification
Response: Checks if user exist in DB

Get all Genres

Method: GET
URL Example: https://ws-node-bluzit-service.herokuapp.com/get_all_genres
Response: gets and display all Genres from DB

Get User Albums By Genres

Method: GET
Required Parameters: username: <string>
URL Example: https://ws-node-bluzit-service.herokuapp.com/get_user_albums_by_genres/maya
Response: gets and display all User Albums by picked Genres from DB

Get User Personal Data

Method: GET
Required Parameters: username: <string>
URL Example: https://ws-node-bluzit-service.herokuapp.com/get_user_personal_data/maya
Response: gets all User general info from DB

Get Album

Required Parameters: album_id <Number>
URL Example: https://ws-node-bluzit-service.herokuapp.com/get_album/4
Response: get all the requested album data from DB

Get User Album

Method: GET
Required Parameters: username: <string>
URL Example: https://ws-node-bluzit-service.herokuapp.com/get_user_albums/maya

Get User Genres

Method: GET
Required Parameters: username: <string>

Get All Albums

Method: GET
URL Example: https://ws-node-bluzit-service.herokuapp.com/get_all_albums
Response: gets and display all Albums from DB

Status for success function

When you use functions that connecting to DB you will get a status JSON message.
{
    "status": "true",
    "message": "Genres(New Orleans Blues, Delta Blues) was added to 'maya'"
}

Error Handling

If URL Address is incorrect you will get a friendly 404 Page.
When you use functions that connecting to DB you will get a status JSON message.
{
    "status": "false",
    "message": "reset genres unsucceeded",
    "error": [
        {
            "name": "parameters",
            "message": "Wrong Genre name"
        }
    ]
}
